# kyujipy
**kyujipy** is a  Python library to convert Japanese texts from Shinjitai (新字体) to Kyujitai (舊字體) and vice versa

**kyujipy** is largely based on the [kyujitai.js](https://github.com/hakatashi/kyujitai.js) project.
